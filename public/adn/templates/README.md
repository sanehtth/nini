# ADN Templates (Option B)

Đây là thư mục template dùng chung cho nhiều dự án/xóm.

- style_templates.json: preset style để UI load và fill form.
- background_templates.json: (placeholder) preset background.
- outfit_templates.json: (placeholder) preset outfit.

Đặt nguyên thư mục `public/adn/templates/` vào repo để Vercel serve như static.
